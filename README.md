# 项目概述与核心定位
BabyCare 智能婴儿无感守护系统是全球首款集“0分贝绝对静音追踪”、“米级非接触式医用测温”与“端云协同端侧防伪/AI终审”于一体的消费级高端母婴安全硬件。

针对现代新生儿家庭及高端托育机构中，传统看护设备“只监不护”、“频繁误报”、“电机噪音惊扰婴儿”以及“红外测温距离受限”等核心痛点，本系统通过硬核的嵌入式算法与仿生学设计，实现了对婴儿体温、睡姿、窒息风险的全天候、无感化、高精度主动式守护，更能全天候监测婴儿床周边的空气质量，如：甲醛、二氧化碳浓度、灰尘（过敏）、温湿度等，甚至包括隐形噪音，为新生儿家庭及高端托育机构提供一站式、全维度的婴儿健康空间防御护盾。

## 主要功能
- 监测空调房环境温与宝宝体表温度变化；
- 检测室内空气质量（pm指数、co2浓度、噪音、甲醛等）
- 提示踢被子着凉
- AI 大模型辅助观察睡姿习惯

# 1. 呼吸健康防御：微生态(婴儿房)空气质量监测阵列
新房装修残留、夜间紧闭门窗导致的二氧化碳超标，是诱发婴儿发育迟缓、呼吸道过敏及深层昏睡风险的隐形杀手。
- **技术闭环：** 系统集成高精度传感器阵列，包括电化学甲醛传感器、二氧化碳（CO2）监测仪、激光散射PM2.5颗粒计数器以及数字级温湿度芯片。
- **用户价值：** 实时捕捉婴儿床边厘米级的空气异动。当夜间二氧化碳（CO2）浓度过高或甲醛有释出抬头趋势时，系统将通过物联网看板第一时间向父母预警，提醒开窗或联动智能家居开启新风，守护婴儿脆弱的呼吸系统。


# 2. 米级医用非接触测温：MLX90614-DCI 窄视角矩阵
- **技术闭环：**  区别于市面上由于视场角（FOV）过大、挂远了测不准的普通红外热成像（如 AMG8833/MLX90640），本系统核心标配 MLX90614-DCI（5°窄视角）医用级传感器，通过金属长管物理聚焦。
- **用户价值：** 在距离婴儿床 1.5米 的远端安全距离外，其有效测温区域能精准锁死在约8.7厘米（刚好覆盖婴儿额头范围），刷新率经算法深度优化至4Hz，在保障 $\pm0.14^\circ\text{C} \sim \pm0.18^\circ\text{C}$ 极端医用精度的同时，，配合温湿度传感器进行环境温动态补偿，实现踢被子着凉或发烧的秒级丝滑报警。

# 3. 睡眠噪声量化：高灵敏度音频环境监测
婴儿在浅睡眠期极易受到外界突发噪声或持续低频噪声（如楼顶噪音、大路车流、电器共振）的惊扰，导致生长激素分泌受到抑制。
- **技术闭环：**  板载高信噪比驻极体/MEMS 麦克风与音频前段处理电路或噪声传感器，实时进行声压级（dB）量化与频谱分析。
- **用户价值：** 精准区分环境背景噪音与宝宝自身的哭闹啼哭。当房间噪声超过设定的婴儿舒适阈值时，自动记录并推送报警，帮助父母科学优化婴儿房的隔音环境。

# 4. 0分贝绝对静音追踪：磁场定向控制云台（FOC）
- **技术闭环：**  拒绝传统机械舵机多齿轮带来的磨齿噪声。产品采用微型两自由度无刷云台电机，引入 SimpleFOC（磁场定向控制）闭环算法，将电流的 PWM 驱动频率强推至 $32\text{ kHz}$ 超声波频段（超越人类及婴儿听觉上限）。
- **用户价值：** 即使在深夜最安静的婴儿房内，云台进行动态视觉追踪与体温扫描时，机械噪音完全为 0分贝。绝不惊醒浅睡眠期的婴儿。

# 5. “零误报”智能视觉：端云协同多模态 AI 终审架构
- **技术闭环：** 硬件端利用轻量化模型进行秒级动态卡哨检测；一旦捕获疑似“趴睡窒息”或“面部遮挡”异常，立即通过低延时物联网骨干链路上传至云端多模态大模型进行视觉终审。
- **用户价值：** 彻底终结了传统摄像头由于衣服褶皱、被子隆起带来的“全天候误报轰炸”。大模型具备常识理解力，能100%分清“褶皱的被子”与“真实的窒息风险”，将夜间误报率直接降低至接近0%。
